## Dependencies
* NPM
* Bower
* Grunt

## Usage

### Install/Update
```sh
$ npm install
$ bower install
```

### Build
```sh
$ grunt
```

### Development
```sh
$ grunt server
```

### Deploy
```sh
$ cd deploy
$ cap production deploy
```

## Supported browsers
* IE9+
* Chrome 4+
* Firefox 4+
* Safari 5+
* Opera
