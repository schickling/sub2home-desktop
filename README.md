## Dependencies
* NPM
* Bower (`npm install -g bower`)
* Grunt (`npm install -g grunt-cli`)

## Usage

#### Install & Update
```sh
$ npm install
$ bower install
```

#### Development / Server
Starts webserver on `localhost:8081` with livereload.
```sh
$ grunt server
```

#### Build (just needed for deployment)
```sh
$ grunt build
```

## Supported browsers
* IE9+
* Chrome 4+
* Firefox 4+
* Safari 5+
* Opera
