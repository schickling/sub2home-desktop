## Dependencies
* NPM
* Bower
* Grunt

## Usage

#### Install & Update
```sh
$ npm install
$ bower install
```

#### Build
```sh
$ grunt build
```

#### Development
```sh
$ grunt server
```

## Supported browsers
* IE9+
* Chrome 4+
* Firefox 4+
* Safari 5+
* Opera
