## Dependencies
* NPM
* Bower
* Grunt

## Install/Update
```sh
$ npm install
$ bower install
```

## Build
```sh
$ grunt
```

## Development
```sh
$ grunt server
```

## Deploy
```sh
$ cd deploy
$ cap production deploy
```
